## 実行手順

以下のコマンドを Bash ツールで実行し、出力をそのままユーザーに表示する。

```bash
cd backend && php import.php 2>&1
```

タイムアウトは 120000ms（2分）を指定すること。

## 出力の読み方

- `完了: XXXXX 件インポート（スキップ: YYY 件）` — 正常終了
- `ERROR:` で始まる行 — エラー。内容をユーザーに伝える
- スキップ件数は、フォーマット不正行・1文字語などの除外分であり正常な動作

## 注意

- このコマンドは既存の words.db を**削除してから**再作成する（冪等操作）
- 実行前にユーザーへの確認は不要（ユーザーが明示的に依頼している前提）
