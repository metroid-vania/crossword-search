## 実行手順

1. ユーザーのメッセージまたはスラッシュコマンドの引数から**検索パターン**を取り出す
2. 以下のコマンドを Bash ツールで実行する（タイムアウト 30000ms）：

```bash
php "C:\Users\sideroll\My Projects\crossword-search\backend\search_cli.php" "<パターン>" 2>&1
```

3. 出力をそのままユーザーに表示する

## パターン記法

| 記号 | 意味 |
|------|------|
| `？` または `?` | 任意の1文字 |
| `＊` または `*` | 任意の0文字以上 |
| `１`～`９` | ナンクロ数字（同じ数字=同じ文字、違う数字=違う文字） |
| ワイルドカードなし | 完全一致 |

## 注意

- パターンに半角スペースが含まれる場合はクォートで囲む（シェルのエスケープ対策）
- 200件を上限として表示し、それ以上ある場合は「200件まで表示」と案内される
